# Core Module

This module contains core functionality used across the app:

- `constants/` - App constants (colors, strings, endpoints, etc.)
- `theme/` - AppTheme, color schemes, typography
- `services/` - API clients, storage services, etc.
- `utils/` - Helper functions, extensions, validators
- `errors/` - Error handling classes
- `usecases/` - Base use case classes
