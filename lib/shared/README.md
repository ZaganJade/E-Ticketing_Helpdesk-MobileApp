# Shared Module

This module contains reusable components shared across features:

- `widgets/` - Reusable widgets (buttons, inputs, cards, etc.)
- `components/` - Higher-level composite components

Components follow the design system defined in the spec:
- AppButton (primary, secondary, destructive, ghost, outline)
- AppInput (text, password, textarea)
- AppCard (ticket, comment, stat, notification)
- StatusBadge (TERBUKA, DIPROSES, SELESAI)
- AppModal (confirmation, form)
- Toast notification
- Skeleton loading
